# My Web Dev Journey: Class 4 Notes
## Headings, Paragraphs, and My First "Utility" Site

Today I leveled up my HTML skills and built something actually useful—a Bookmark Manager! I also learned some "ninja techniques" in VS Code that make me feel like a much faster coder. Here’s what I learned.

---

### 1. The Heading Hierarchy (h1 to h6)
I learned that headings aren't just about making text big or small—they are about **hierarchy**.
*   **`<h1>`**: My main title. I should usually only have one per page.
*   **`<h2>` to `<h6>`**: These are for sub-sections.
*   **Why it matters:** Browsers and search engines (like Google) use these tags to understand the structure of my content. Using them correctly is part of being a "Sigma" developer.

---

### 2. Paragraphs & The Lorem Trick
I used the `<p>` tag for regular text. But the best part was learning how to generate "dummy text" instantly:
*   If I type `lorem23` and hit Enter, VS Code gives me 23 random words. 
*   This is perfect for filling up a page while I’m still designing it. It's much better than typing "test test test" over and over!

---

### 3. VS Code Ninja Techniques
I learned two shortcuts that boosted my productivity:
*   **Multiple Cursors:** If I hold `Alt` and click in different places, I can type in multiple spots at the same time.
*   **Fast Duplicate:** `Alt + Shift + Down Arrow` duplicates the current line instantly.

---

### 4. Anchors & Links (The `<a>` Tag)
This is the core of the web! I learned how to link to other sites:
*   **`href`**: The attribute where I put the URL (e.g., `href="https://google.com"`).
*   **`target="_blank"`**: I used this to make links open in a **new tab** so people don't leave my website when they click a link.
*   **Visited Links:** I noticed that links I've already clicked turn purple automatically. That's the browser helping the user out!

---

### 5. My Bookmark Manager Project
I built a `bookmarkManager.html` file. Instead of just using Chrome's bookmarks, I made my own! 
*   I used `<h1>` for the top title.
*   I used `<h2>` and `<h3>` to categorize my "Main" and "Secondary" bookmarks.
*   I used `<p>` tags to separate my links so they don't all sit on one line.
