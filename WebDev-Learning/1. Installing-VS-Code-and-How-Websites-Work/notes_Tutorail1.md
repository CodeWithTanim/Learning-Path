### Installing my main tool: VS Code
I learned that I need a "Source Code Editor" to write code. While I could use Notepad, everyone uses **VS Code**. 

**What I did to install it:**
*   I went to the official site and downloaded the installer for Windows.
*   **Important part I noticed:** When installing, I had to check the boxes that say **"Add 'Open with Code'"**. This is really handy because now I can just right-click any folder and open it instantly in VS Code.
*   I chose the **Dark Modern** theme because it looks cool and is easy on the eyes.

**Some cool shortcuts I found:**
*   **Auto Save:** I turned this on (`File > Auto Save`) so I don't have to keep hitting save every time I type a letter.
*   **Zoom with Mouse:** I enabled "Mouse Wheel Zoom" in settings. Now I can just hold `Ctrl` and scroll to make the text bigger.
*   **The '!' Trick:** If I type `!` and hit Enter in an HTML file, it gives me all the basic code I need to start. Super fast!

### How I understood the "Internet"
The teacher explained how websites work using a restaurant analogy, and it finally clicked for me:

*   **Me (The Client):** I'm the customer sitting at the table.
*   **The Browser (The Waiter):** This is Chrome or Brave. It takes my order (the URL I type) and brings me the website once it's ready.
*   **The Server (The Kitchen):** This is where all the website files are stored. The kitchen "cooks" the page and sends it back to the waiter.

**Response Codes I need to remember:**
*   **200:** Everything went perfect (Food is served!).
*   **404:** The page doesn't exist (Menu item not found).
*   **500:** Something broke on the server (Kitchen fire!).

---

### HTML, CSS, and JavaScript: The Big Three
I learned that every website I see is made of these three things working together. Here is how I'm thinking about them:

1.  **HTML (The Skeleton):** It's just the raw structure. Like the bones of a person or the frame of a car.
2.  **CSS (The Style):** This is the beauty. It’s the clothes, the paint on a car, or the UI colors. It makes things look pretty.
3.  **JavaScript (The Brain):** This makes things actually *do* stuff. It's the engine in a car or the human brain. Clicking buttons, animations, and logic happen here.
